# Falden Clothing - Next.js E-commerce Store

A modern, responsive, and feature-rich e-commerce store built with **Next.js**, **Tailwind CSS**, and **TypeScript**. Includes product listings, detailed product pages, shopping cart, favorites, and customer reviews. Designed with **glassy-morphic UI elements** and dark/light mode support.

---

## Features

- **Product Catalog**: Browse all products with categories and featured highlights.
- **Product Detail Page**: Includes images, ratings, detailed HTML content, and customer reviews.
- **Shopping Cart**: Add, remove, and adjust quantities.
- **Favorites**: Mark products as favorites and view them separately.
- **Responsive Design**: Works perfectly on mobile, tablet, and desktop.
- **Glassy-morphic Cards**: Modern UI for product info and reviews.
- **Dark/Light Mode**: Tailwind CSS classes support both modes.
- **Static Export Support**: Fully pre-rendered pages using `generateStaticParams` for `/product/[id]`.
- **Related Products**: Suggested products on each product page.

---

## Tech Stack

- [Next.js 13](https://nextjs.org/) (App Router)
- [Tailwind CSS](https://tailwindcss.com/)
- [TypeScript](https://www.typescriptlang.org/)
- [Framer Motion](https://www.framer.com/motion/)
- [Lucide React Icons](https://lucide.dev/)
- Context API for Cart & Favorites management
